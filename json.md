# XML
* The reference to entity "xxx" must end with the ';' delimiter
  * solution; use escaping
    * `" -> &amp;quot;`
    * `' -> &amp;apos;`
    * `< -> &amp;lt;`
    * `> -> &amp;gt;`
    * `& -> &amp;amp;`
